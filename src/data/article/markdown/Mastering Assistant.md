 # Dynamics
[[LUFS]]
Loudness Units Full Scale
人間の耳は大きい音ほど良い音と認識するという前提
昔は[[RMS]]
- Integrated LUFS
楽曲全体の平均音圧のこと
- ラウドネスノーマライゼーション
 LUFS/LKFSという単位を基準に、ある一定値以上の音圧の高い音源は自動で音量を下げるシステム
ほとんどの配信サービスは-14.0LUFS
（SoundCloudは-8.0LUFS）

ただし圧縮感もジャンルによっては必要

参考
- EDM : -6～-8LUFS
- HipHop : -8～-11LUFS
- Trap Beat : -12～-14LUFS
- Rock : -7～-9LUFS
- Metal Core : -4～-6LUFS
