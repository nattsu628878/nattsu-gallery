イテレーター系
1. collect()メソッド
	生成する型としてStringやResultを指定できる
2. enumerate()メソッドとzip()メソッド
	- enumerate() - リストの要素とイテレーション数とが同時に必要な時に使える
	- zip() - そのイテレーターの要素と別のイテレーターの要素を同時に一つずつ走査（一つの事柄を複数のリストに分けて管理しているときに内容を同時に取り出したいときに使える） 
パターンマッチ系（match構文）
1. if-let let-else
	パターンマッチを使って値を取り出すと同時に処理を分岐する。
	OptionがSomeかどうかを確認して、Someであれば中の値を取り出す
2. matches!()マクロ
	値がパターンに一致するかを確認し、その結果をboolで返すマクロ（match構文と一緒）条件式に使う用。見た目的にmatches!()マクロのほうがいいかも？
```rust
fn main(){
	let num = 7144;
	if [7, 71, 714, 7144].contains(&num){
		println!("hit!");
	}
	
	let num = 7144;
	if matches!(num, 7 | 71 | 714 | 7144){
		println!("hit!");
	}
}
```
標準ライブラリ
1. std::memのtake() replace()
	引数に値を要求する関数には構造体のフィールドを直接適用することはできないので、一度フィールドから値として取り出してから適用しなければならないが、これらで簡単に取り出せる。元の変数には、take()は型の初期値、replace()は指定した値が入る。
2.  std::num::NonZero;
	変数に0が入らないことを保証できる。データの異常値を0で表現せずに、OptionやResultで表現するときに使える。
