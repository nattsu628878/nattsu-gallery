# PCガイド for iPhone ユーザー

## はじめに：なぜPCなのか

### PCとは何か

PCは**Personal Computer**の略で、あなた個人専用の情報処理装置です。iPhoneも広義のコンピュータですが、PCはより自由度が高く、創造的な作業に特化しています。

### iPhoneとPCの違い

| 項目      | iPhone         | PC (Mac)         |
| ------- | -------------- | ---------------- |
| 主な用途    | 情報消費・コミュニケーション | 情報創造・生産活動        |
| 画面      | 小さい（片手サイズ）     | 大きい（複数ウィンドウ表示可能） |
| 入力      | タッチ・フリック       | キーボード・マウス（精密操作）  |
| ファイル管理  | アプリ内に隠蔽        | 自由に管理可能          |
| カスタマイズ性 | 制限あり           | 高い自由度            |
| できること   | アプリ依存          | プログラミング・開発・制作    |

## 1. Mac を選ぶ・購入する

### 1.1 Mac の種類

### MacBook（ノート型）

- **MacBook Air**：軽量・長時間バッテリー・静音（初心者におすすめ）
- **MacBook Pro**：高性能・プロ向け（動画編集・音楽制作・開発）

### iMac（デスクトップ型）

- 大画面一体型
- 持ち運ばない人向け

### Mac mini / Mac Studio

- モニター別売り
- 上級者向け

### 1.2 スペックの選び方

### チップ（頭脳）

- **M3/M4チップ**：最新・高性能（おすすめ）
- **メモリ（RAM）**：16GB以上（複数アプリを同時に使う）
    - = 動作の快適さ度合い
- **ストレージ（SSD）**：512GB以上（写真・動画・音楽を保存）
    - アプリ自体のデータを保存するため（↑のようなデータはクラウドに）

### 1.3 購入場所

- **Apple Store**（公式、学割あり）
- **家電量販店**（ポイント還元）
- **Amazon**（セール時お得）

### 1.4 一緒に買うべきもの

- **USB-Cハブ**（USB-A、HDMI、SDカードなど拡張）
- **外付けSSD**（バックアップ用、1TB程度）
- **マウス**（Magic Mouse or ロジクール MX Master）
    - トラックパッドが優秀なのでいらないかも
- **外部モニター**（作業効率アップ、23〜27インチ）
    - いずれ買えばよし

## 2. 初期設定

### 2.1 開封〜起動

1. 箱を開けて本体を取り出す
2. 電源ボタンを長押し（Touch ID部分）
3. 画面の指示に従う

### 2.2 Apple ID でサインイン

- iPhoneで使っているApple IDを入力
- 自動的にiCloudと同期される
- 写真・連絡先・メモがMacでも使える

### 2.3 Touch ID（指紋認証）の設定

- ロック解除、支払い認証に使用
- iPhoneのTouch IDと同じ

### 2.4 Trackpad（トラックパッド）の設定

- **システム設定 > トラックパッド**
- タップでクリック：オン（推奨）
- スクロール方向：ナチュラル（iPhoneと同じ）

### 2.5 Dock（ドック）の設定

- 画面下のアプリバー（iPhoneのホーム画面に相当）
- よく使うアプリを並べる
- 不要なアプリは削除（ドラッグ&ドロップで外す）

## 3. 基本操作（iPhoneとの対応）

### 3.1 マルチタッチジェスチャー

| 操作 | iPhone | Mac（トラックパッド） |
| --- | --- | --- |
| タップ | タップ | 1本指でタップ |
| スクロール | 上下スワイプ | 2本指で上下スワイプ |
| 戻る | 左スワイプ | 2本指で左スワイプ |
| ホームに戻る | 下からスワイプ | 親指と3本指でピンチ |
| アプリ切り替え | 上スワイプ長押し | 3本指で左右スワイプ |
| 拡大・縮小 | ピンチイン/アウト | 2本指でピンチイン/アウト |

### 3.2 Finder（ファイル管理）

- iPhoneの「ファイル」アプリの強化版
- 全てのファイルを自由に管理できる
- **Command + N**：新規ウィンドウ
- **Command + T**：新規タブ

### 3.3 メニューバー

- 画面最上部の操作メニュー
- 右上：Wi-Fi、バッテリー、日時、Spotlight（検索）など
- 左上：アプリ名、ファイル、編集、表示など

### 3.4 Spotlight 検索

- **Command + Space**で起動
- アプリ、ファイル、設定、計算、単位変換など何でも検索
- iPhoneの検索機能の強化版

### 3.5 Mission Control（マルチタスク）

- **3本指で上スワイプ**
- 開いている全ウィンドウを一覧表示
- デスクトップ（作業スペース）を複数作成可能

3. **ダウンロードは定期的に整理**（1週間以内に移動or削除）

## 8. ターミナル（黒い画面）の基礎

### 8.1 ターミナルとは

- コンピュータに文字で命令を出すツール
- GUI（グラフィカル）では できない操作が可能
- 開発者・研究者の必須スキル

### 8.2 ターミナルを開く

- **Applications > ユーティリティ > ターミナル**
- または **Command + Space → “ターミナル”と入力**

### 8.3 基本コマンド

```bash
# 現在のディレクトリを表示pwd# ファイル一覧を表示lsls -la  # 詳細表示# ディレクトリ移動cd Documents  # Documentsフォルダへcd ..         # 一つ上のフォルダへcd ~          # ホームディレクトリへ# フォルダ作成mkdir my_project
# ファイル作成touch test.txt
# ファイル削除rm test.txt
# フォルダ削除rm -r my_project
# ファイル内容表示cat test.txt
# コマンド履歴history
```

### 8.4 パッケージマネージャー（Homebrew）

```bash
# Homebrewインストール（公式サイトのコマンドをコピペ）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# アプリのインストール
brew install node
brew install python

# アプリ検索brew search <アプリ名>
# インストール済み一覧brew list
```

## 9. トラブルシューティング

### 9.1 動作が遅い

- **再起動**：Command + Control + 電源ボタン
- **メモリ確認**：アクティビティモニタで確認
- **ストレージ確認**：Apple メニュー > このMacについて > ストレージ
- **不要なアプリ削除**
- **BuhoCleanerでクリーニング**

### 9.2 アプリが動かない

- **強制終了**：Command + Option + Esc
- **再インストール**：アプリ削除→再インストール
- **macOS更新**：最新版にアップデート

### 9.3 Wi-Fiに繋がらない

- **Wi-Fiオフ/オン**
- **ルーター再起動**
- **システム設定 > ネットワーク > 詳細 > DNSをリセット**

### 9.4 バックアップ

- **Time Machine**：外付けHDDで自動バックアップ
- **iCloud**：重要ファイルをクラウド保存
- **手動コピー**：外付けSSDに定期的にコピー

## 11. まとめ：PCとの付き合い方

### 11.1 PCは「道具」である

- 使いこなすには練習が必要
- 毎日少しずつ触ることが上達の鍵
- 失敗を恐れない（元に戻せる）

### 11.2 インプットとアウトプットのバランス

- **情報収集ばかりにならない**
- **実際に手を動かして作る**
- **作品を公開して フィードバックをもらう**

### 11.3 学習の心構え

1. **好奇心を持つ**：“これってどうやるんだろう？”
2. **検索力を磨く**：“Mac ○○ やり方” で検索
3. **失敗から学ぶ**：エラーは成長のチャンス
4. **継続する**：毎日10分でもPCに触る

### 11.4 最初の1ヶ月でやること

- [ ]  Macの基本操作に慣れる

## 付録：用語集

- **GUI**（Graphical User Interface）：グラフィカルなインターフェイス（アイコン・マウス操作）
- **CLI**（Command Line Interface）：文字ベースのインターフェイス（ターミナル）
- **ディレクトリ**：フォルダのこと
- **パス**：ファイルの住所（例：/Users/username/Documents/file.txt）
- **拡張子**：ファイル名の最後の「.txt」「.jpg」など
- **RAM**：メモリ、作業机の広さ
- **SSD**：ストレージ、引き出しの容量
- **CPU/チップ**：頭脳、処理速度
- **バックアップ**：データのコピー（保険）
- **クラウド**：インターネット上のストレージ
- **オープンソース**：無料で使えるソフトウェア
- **プラグイン**：機能拡張（アプリの中のアプリ）
